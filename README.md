# Cilento Collection · Guest Guide

## File inclusi

- `index.html`: pagina principale
- `style.css`: stile della pagina
- `guest-guide.pdf`: va aggiunto o sostituito con la Guest Guide aggiornata

## Cosa modificare prima della pubblicazione

Apri `index.html` e sostituisci:

1. `https://maps.google.com` con il link Google Maps corretto.
2. `https://wa.me/39XXXXXXXXXX` con il numero WhatsApp completo, senza spazi.
3. `https://www.airbnb.it` con il link dell'annuncio o della pagina recensioni.

## Come aggiornare la guida senza cambiare QR

Ogni nuova versione deve essere caricata con lo stesso nome:

`guest-guide.pdf`

Su GitHub:

1. apri il file vecchio;
2. eliminalo oppure carica quello nuovo sovrascrivendolo;
3. verifica che il nuovo PDF si chiami esattamente `guest-guide.pdf`.

Il link della pagina e il QR resteranno invariati.
